Projeto Elo Solidario

Elo Solidário 🤝
O Elo Solidário é uma plataforma desenvolvida para centralizar e agilizar o suporte em situações de emergências e desastres ambientais. O projeto atua em três frentes críticas: localização de pessoas desaparecidas, gestão de abrigos de emergência e mobilização de voluntários e doações.

🚀 Sobre o Projeto
Em cenários de crise, a dispersão de informações dificulta o socorro. O Elo Solidário une tecnologia e solidariedade para criar uma rede de amparo eficiente, permitindo que abrigos gerenciem suas capacidades e que a ajuda chegue onde é mais necessária.

Funcionalidades Principais
Gestão de Abrigos: Cadastro e monitoramento de abrigos temporários e pessoas acolhidas.

Central de Voluntariado: Conexão entre instituições que precisam de auxílio e pessoas dispostas a ajudar.

Busca de Desaparecidos: Banco de dados otimizado para auxiliar no reencontro de familiares.

Logística de Doações: Listagem em tempo real das necessidades mais urgentes de cada unidade.

🛠️ Tecnologias Utilizadas
O projeto utiliza uma stack moderna focada em performance e escalabilidade:

Frontend: React com SASS/SCSS para uma interface responsiva e intuitiva.

Backend: Node.js com Express para uma API ágil e segura.

Banco de Dados: SQLite para persistência de dados e gerenciamento de tabelas de incidentes, abrigos e usuários.

Design: Inkscape para criação de identidade visual e elementos gráficos.

📦 Como Instalar e Rodar
Clone o repositório:

Bash
git clone https://github.com/seu-usuario/elo-solidario.git
Configure o Backend:

Bash
cd backend
npm install
npm start
Configure o Frontend:

Bash
cd frontend
npm install
npm start
🗄️ Estrutura do Banco de Dados
O sistema conta com um esquema de banco de dados robusto para garantir a integridade das informações:

abrigos: Informações sobre localização e capacidade.

abrigados: Registro detalhado de pessoas acolhidas para facilitar buscas.

incidentes: Relatos de ocorrências e necessidades imediatas.
